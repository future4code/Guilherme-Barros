Alguns dos meus projetos desenvolvidos até o momento

Os projetos em questão utilizam linguagens de marcação como HTML,CSS e programação como PHP e Javascript.

1

Calculadora IMC

O índice de massa corporal é uma medida internacional usada para calcular se uma pessoa está no peso ideal. Trata-se de um método fácil e rápido para a avaliação do nível de gordura de cada pessoa.
A calculadora de Índice de Massa Corporal foi desenvolvida em HTML 5, CSS 3 e Javascript puros.

2

Ast3roid

Ast3roid é um jogo 32 bits em modo arcade desenvolvido para a feira de ciências da Etec Zona Leste em 2019. O jogo consiste em uma nave que passeia pelo espaço e sua missão é desviar ao máximo dos asteroides que vão caindo de forma randomica na tela.
Ast3roid foi desenvolvido em HTML, CSS, Javascript e PHP.

3

Clone Instagram

O Clone Instagram é um projeto desenvolvido para um bootcamp de desenvolvimento web do curso de HTML intermediário ministrado pela Digital Innovation One. Trata-se da tela inicial de login do Instagram.
O projeto foi desenvolvido com HTML e CSS.
