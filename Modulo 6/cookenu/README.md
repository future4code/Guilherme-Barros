# COOKENU

## Primeiros Passos

* Clonar este repositório
* Executar `npm install` para adicionar as dependências
* Criar um arquivo .env na raiz do projeto e preencher as chaves a seguir com os valores apropriados:
   ```
  DB_HOST=
DB_USER=
DB_PASSWORD= 
DB_NAME=
BCRYPT_COST=
JWT_KEY=
NODEMAILER_USER=
NODEMAILER_PASS=
 
   ```
